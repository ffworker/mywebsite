Vorbereitung

Falls Docker noch nicht vorhanden ist, installieren SieDocker.

For good practice: run the following commands in the /home of the user you are using. e.g from /home/user01

Create Folders for the docker-container to store locally outside of the container:

mkdir /opt/influxdb
mkdir /opt/grafana

give full access to the folders just created

chmod 775 /opt/influxdb/ /opt/grafana/

Now Start the Influxdb(v1.8) - Container

Docker run -p 8086:8086 \ 
-v $PWD:/var/lib/influxdb
Influxdb:1.8
